# semana3-html-css

Ejercicio de la semana 3 — Desarrollo Web.

Replica básica de una noticia online utilizando **HTML semántico** y **CSS** para emular la apariencia del diario original.

## Archivos

- `Noticia.html` — estructura HTML de la noticia.
- `style.css` — hoja de estilos.

## Noticia original

| Diario | URL |
|---|---|
| Chequeado | https://chequeado.com/ultimas-noticias/manuel-adorni-frente-al-congreso-todos-los-chequeos-a-su-discurso/ |
